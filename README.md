Projeto de **análise de dados em Python** desenvolvido durante a **Imersão de Dados da Alura**, com foco em **exploração, tratamento e visualização de dados**, utilizando **Google Colab**.

Este repositório faz parte do meu **portfólio técnico**, demonstrando habilidades práticas em **Data Analysis**, organização de projetos e boas práticas em notebooks.

---

## Sobre o Projeto

Ao longo da imersão, trabalhei com **dados reais**, aplicando técnicas amplamente utilizadas no mercado para:

- Entender o contexto dos dados  
- Identificar padrões e inconsistências  
- Limpar e preparar dados para análise  
- Gerar visualizações claras e orientadas a insights  

O objetivo foi simular um **fluxo real de trabalho de um analista de dados**, desde a exploração inicial até análises finais.

---

## Tecnologias Utilizadas

- **Python**
- **Google Colab**
- **Pandas** – manipulação e análise de dados  
- **NumPy** – operações numéricas  
- **Matplotlib** – visualização de dados  
- **Seaborn** – visualizações estatísticas

## Como Executar

### Executar no Google Colab (Recomendado)

1. Acesse este repositório no GitHub  
2. Abra o notebook desejado (`.ipynb`)  
3. Clique em **“Open in Colab”**  
4. Execute as células diretamente no navegador  

> Não é necessário instalar dependências localmente.

---

## Execução Local

### 1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/imersao-dados-alura.git
```

### 2. Acesse o diretório do projeto:
```bash
cd imersao-dados-alura
```

### 3. Instale as dependências necessárias:
```bash
pip install pandas numpy matplotlib seaborn
```

### 4. Execute os notebooks:
```bash
jupyter notebook
```
